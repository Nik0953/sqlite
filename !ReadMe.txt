Создаем базу данных вакансий с hh.ru,
а затем просматриваем ее через web-интерфейс, написанный на Flask


*******************************************************************
    Отдельно однократно запускается модуль подготовки данных:
    'prepare_data_app.py'
    он получает данные вакансий из api hh.ru
    и затем создает таблицы в файле 'vac_hh.sqlite'

        Создаются таблицы:
            Вакансии:       vacancy
            Регионы:        region   (из ранее созданного файла  '/data/regions.json' )
            Валюты:         currency
            Ключевые слова: keywords (из заранее подготовленного списка)
